Word AI Add-in - Local Install
==============================

HOW TO USE:

1. Run "Start-WordAI.bat"
   - Double-click it
   - A window will open - KEEP IT OPEN

2. In Microsoft Word:
   - Insert -> Add-ins -> Upload My Add-in
   - Click "Browse" and select manifest.xml from this folder
   - Click "Add"

3. Use the add-in:
   - Go to the Home tab
   - Click "AI Assistant"
   - Enter your OpenAI API key (get one at platform.openai.com)
   - Start chatting to edit your document!

REQUIREMENTS:
- Microsoft Word (desktop)
- Node.js (https://nodejs.org) OR Python (https://python.org)
- OpenAI API key

SHARING:
- Zip this entire folder and send it
- Recipients follow the same steps above
